# Optimus Users

## Installation

```bash
composer require optimuscms/users
```

## Usage

### Api routes

```http
GET /admin/api/users
```

```http
POST /admin/api/users
```

```http
GET /admin/api/users/{id}
```

```http
GET /admin/api/user
```

```http
PATCH /admin/api/users/{id}
```

```http
DELETE /admin/api/users/{id}
```
