# Optimus Users

## Installation

```bash
composer require optimuscms/users
```

## Usage

```http
GET /admin/users
```

```http
POST /admin/users
```

```http
GET /admin/users/{id}
```

```http
GET /admin/user
```

```http
PATCH /admin/users/{id}
```

```http
DELETE /admin/users/{id}
```
